Finalizing Setup

For support in English, ask for help at HENkaku on Discord.

# Required Reading

We will now setup the following application:

  • iTLS InstallerOpen in new window (adds TLSv1.2 support in older firmwares for online connection)

In order to install the application on your device, we use the VitaDeploy application to download the latest version of the app.

# What You Need

  • The VitaDeploy application installed on your device
    • If you don't have VitaDeploy installed, follow Installing VitaDeploy

# Instructions

# Section I - Installing homebrew applications

  1. Launch the VitaDeploy application
  2. Tap "App downloader"
    • If you don't see a green menu with this option, make sure you've enabled unsafe homebrew
  3. Select the following application:
    • iTLS Installer
  4. Tap "Download the selected apps"
  5. Wait for the app to finish downloading and installing
    • When finished, you exit VitaDeploy

# Section II - Installing iTLS

  1. Open the iTLS-Enso application
  2. Press × on "Install the full iTLS package"
    • If not on HENkaku Ensō, select "Install the iTLS compat module (web-browser)" instead
  3. Wait for your device to reboot

# Section III - Blocking Updates

  1. Launch the Settings application
  2. Navigate to System -> Auto-Start Settings
  3. Uncheck the Download Update File for System Software
  4. Close the Settings application

# Section IV - PSN Access

  1. Launch the Settings application
  2. Navigate to HENkaku Settings
  3. Check "Enable PSN spoofing"
  4. Check "Enable Version Spoofing"
  5. Navigate to "Spoofed Version"
  6. Enter "3.74" into the box
    • Should a new firmware version be released in the future, you must change the spoofed version to match in order to access PSN
  7. Close the Settings application

DANGER

It is not recommended to make modifications to your Custom Firmware installation by installing homebrew applications intended for advanced users (such as "Enso EX"). Doing so without fully understanding the system can lead to a BRICK!

TIP

For information on replacing your Sony memory card with alternative storage (such as a microSD card or USB drive), check out the YAMT page.
