Installing VitaDeploy

For support in English, ask for help at HENkaku on Discord.

The guide uses the VitaDeploy application as the primary method of installing other apps and a custom firmware. If you didn't install VitaDeploy when installing h-encore, you can install it yourself with molecularShell or VitaShell.

In order to install the necessary .vpk (content package) file on your device, we use the File Transfer Protocol (FTP)Open in new window to copy the files to your device's memory card.

# What You Need

  • An FTP Client such as WinSCPOpen in new window or CyberDuckOpen in new window
    • Alternatively, you can also use the USB transfer feature of VitaShell
  • The latest version of VitaDeployOpen in new window
    • Download the VitaDeploy.vpk file

# Section I - Prep Work

  1. Launch the VitaShell application
  2. Press START to open the VitaShell settings
  3. Press confim on "SELECT button" to change the mode to "FTP"
    • Alternatively, you can leave this setting on "USB" and transfer all files via a USB cable for the remainder of this guide
  4. Press ○ to close the VitaShell settings
  5. Press SELECT to enable FTP access on your device
  6. Open your FTP client on your computer
  7. Enter the IP Address and Port displayed on your device
  8. Using your FTP client, navigate to ux0: -> data/
  9. Transfer VitaDeploy.vpk to the data/ folder
  10. Press ○ on your device to close the FTP connection

# Section II - Installing the application

  1. Launch the VitaShell application
  2. Navigate to ux0: -> data/
  3. Press × on "VitaDeploy.vpk"
  4. Press × to confirm the install
  5. Press × to confirm the install again when prompted

VitaDeploy should now be installed to your home screen.
